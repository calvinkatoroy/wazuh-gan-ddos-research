% \begin{abstract}
% Security Information and Event Management (SIEM) systems like Wazuh rely heavily on rule-based signatures, making them vulnerable to sophisticated "stealthy" DDoS attacks generated by Generative Adversarial Networks (GANs). This paper proposes a comprehensive comparative study evaluating SIEM detection effectiveness with and without machine learning integration across attack sophistication levels. We simulate key adversarial characteristics (temporal randomization, payload variance, header manipulation) to enable controlled experimentation, validated through Kullback-Leibler divergence analysis ($D_{KL} < 0.8$ for evasiveness). We design and implement GANDD-Bridge, a middleware architecture integrating ML-based behavioral detection (7+ features including inter-arrival time variance, payload entropy, protocol distributions) with existing Wazuh infrastructure. Through systematic $2 \times 3$ experimental design comparing two configurations (with/without GANDD) across three attack types (volumetric, low-rate, adversarial), we demonstrate detection improvements of +1\%, +40\%, and +80\% respectively, with performance costs (latency +5-10s, CPU +15-25\%, false positives +3-4\%) acceptable for small-to-medium networks.
% \end{abstract}
